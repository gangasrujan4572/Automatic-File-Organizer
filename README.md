# 📂 Automatic File Organizer

A Python-based automation tool that monitors a folder and automatically organizes files into categorized directories.

## ✨ Features

- 📂 Organizes files automatically
- 🖼️ Images → Images folder
- 📄 Documents → Documents folder
- 📕 PDFs → PDFs folder
- 🎵 Music → Music folder
- 🎬 Videos → Videos folder
- 📦 Archives → Archives folder
- 👀 Real-time monitoring using Watchdog

## 🛠️ Tech Stack

- Python
- Watchdog
- OS
- Shutil

## 🚀 Installation

```bash
pip install -r requirements.txt
python organizer.py
```
